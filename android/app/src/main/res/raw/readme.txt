# Notification Sound Files

This directory should contain the following sound files for prayer notifications:

- `adhan_mecca.mp3`: Adhan sound from Mecca
- `adhan_medina.mp3`: Adhan sound from Medina
- `short_beep.mp3`: A short beep sound for minimal notifications

## Instructions

1. Place the sound files in this directory with the exact names listed above
2. Make sure the files are in MP3 format for Android compatibility
3. Keep the file sizes reasonable (under 1MB each) to avoid performance issues
